使用IDEA intellij 必须掌握的使用技巧！

2019-12-24

工欲善其事，必先利其器。

必备插件安装

  1. representation assistant
  2. fas

Search Everywhere

https://www.jetbrains.com/help/idea/searching-everywhere.html
set keymap type: Mac OS 10.5+
cmd + o class类查找
cmd + shift + o file文件查找
cmd + alt + o symbols特征查找; filter能按路径查找“／”

press shift twice to open the search window,you can search everywhere you know. (include class files symbols and actions).Press double shift again, Idea will select or unselect the Include non-project itemscheckbox.

start typing your query,IDEA lists the results dividing them into sections where your query is found. press cmd + up or down to jump between sections.And you can press tab to switch the scope of your search to classes files symbols or actions.

press alt + up or down to switch history.

you can input everything ,also example: service.redisservice.lget -scope of symbols. So you are not have to rely on mouse so much now.

cmd + E is your friend command.Then idea will lists recent files.(E presents Edited files)

To navigate backwards, press ⌘[. To navigate forward, press ⌘].

To navigate to the last edited location, press ⇧⌘⌫.

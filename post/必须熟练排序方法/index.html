数据结构与算法 —— 应掌握的排序算法

排序算法

选择排序

工作流程

  1. 在未排序序列中找到最小（大）元素，存放到排序序列的起始位置
  2. 从剩余未排序元素中继续寻找最小（大）元素
  3. 然后放到已排序序列的末尾。
  4. 以此类推，直到所有元素均排序完毕。
    如示例图:

代码如下：

//  重复练习.c
//  几种排序方式
//
//  Created by Next on 17/2/11.
//  Copyright © 2017年 Next. All rights reserved.
//
#include <stdio.h>
int main(){
    //选择排序
     int arr[6];
     int len=sizeof(arr)/sizeof(arr[0]);
    for (int i=0; i<len; i++) {
        printf("请输入第%d个数",i+1);
        scanf("%d",&arr[i]);
    }
    for (int i=0; i<len-1; i++) {
        for (int j=i+1; j<len; j++) {
            if (arr[i]<arr[j]) {
                int temp=arr[i];
                arr[i]=arr[j];
                arr[j]=temp;
            }
        }
    }
    printf("我已将你数入的数字从大到小排序成功\n");
    for (int i=0; i<len; i++) {
        printf("%d\t",arr[i]);
    }
    return 0;

冒泡排序

工作流程

  1. 比较相邻的元素。如果第一个比第二个大，就交换他们两个。
  2. 对每一对相邻元素作同样的工作，从开始第一对到结尾的最后一对。3.在这一点，最后的元素应该会是最大的数。
  3. 针对所有的元素重复以上的步骤，除了最后一个。
  4. 持续每次对越来越少的元素重复上面的步骤，直到没有任何一对数字需要比较。
    如图:

代码如下:

#include <stdio.h>
int main(){
    for (int i=0; i<len-1; i++) {
        for (int j=0; j<len-1-i; j++) {
            if (arr[j]<arr[j+1]) {
                int temp=arr[j];
                arr[j]=arr[j+1];
                arr[j+1]=temp;
            }
        }
    }
    printf("主人我已将你输入的数字从大到小排序好了\n");
    for (int i=0; i<len; i++) {
        printf("%d\t",arr[i]);
    }
}

三.二分搜索算法

工作流程

是一种在有序数组中查找某一特定元素的搜索算法。搜索过程从数组的中间元素开始，如果中间元素正好是要查找的元素，则搜索过程结束；如果某一特定元素大于或者小于中间元素，则在数组大于或小于中间元素的那一半中查找，而且跟开始一样从中间元素开始比较。如果在某一步骤数组为空，则代表找不到。这种搜索算法每一次比较都使搜索范围缩小一半。

如图:

代码如下:

include<stdio.h>
int main(){
    //二分法查找有数列中的数字
    int array[6]={12,23,34,45,56,67};
    int min=0;
    int length=sizeof(array)/sizeof(arr[0]);
    int max=length;
    int mid=(max+min)/2;
    int key;
    printf("请输入你要查询的数字");
    scanf("%d",&key);
    while (array[mid]!=key) {
        if (key<array[mid]) {
            max=mid-1;
        }else if(key>array[mid]){
            min=mid+1;
        }
        mid=(max+min)/2;
    }
    printf("主人我找到了!!key%d的下为",mid);
    return 0;
}
